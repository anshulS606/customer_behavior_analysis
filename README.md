# 📊 Customer Shopping Behavior Analysis

## 📌 Overview
This project analyzes customer shopping behavior using **Python, SQL, MySQL, and Power BI** to extract meaningful business insights. It covers the complete data analysis workflow, including data cleaning, exploratory data analysis (EDA), SQL querying, interactive dashboard development, and business reporting.

---

## 📂 Dataset
The dataset contains customer shopping information, including:

- Customer Details
- Product Categories
- Purchase Amount
- Review Ratings
- Shipping Type
- Discount & Promo Usage
- Previous Purchases
- Payment Methods
- Purchase Frequency

---

## 🛠️ Tools & Technologies

- Python
- NumPy
- Pandas
- Matplotlib
- SQL
- MySQL
- Power BI
- Gamma AI (Presentation)

---

## 📊 Project Workflow

- Loaded the dataset in Python.
- Performed data cleaning and preprocessing.
- Conducted Exploratory Data Analysis (EDA).
- Imported the cleaned dataset into MySQL.
- Wrote SQL queries to solve business problems.
- Built an interactive Power BI dashboard.
- Created a business report and presentation using Gamma.

---

## 📈 Dashboard Highlights

- Sales & Revenue Overview
- Customer Segmentation
- Product & Category Performance
- Customer Purchase Trends
- Shipping Analysis
- Interactive Filters & KPIs

---

## 🔍 Key Results

- Identified top-performing products and categories.
- Analyzed customer purchasing behavior and loyalty.
- Compared shipping methods and sales trends.
- Generated actionable business insights through SQL and Power BI visualizations.

---

## 🚀 How to Run

1. Clone this repository.
2. Install the required Python libraries:

```bash
pip install pandas numpy matplotlib sqlalchemy pymysql
```

3. Run the Python notebook/script for data cleaning and EDA.
4. Import the cleaned dataset into MySQL and execute the SQL queries.
5. Open the Power BI (`.pbix`) file to explore the interactive dashboard.
6. View the project report and presentation for summarized business insights.
